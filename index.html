Transparent HATEOAS

Hypestore (HYPErtext STORage Engine) is a storage engine and coordinator for Hypertext. Hypestore provides a frictionless way for applications using HTTP as a wire protocol to persist entity state. It does it's best not to be too picky about what you throw at it, just don't cross RFC 2616. Together, we'll be 200 OK.

Synopsis

For request URL: http://hypestore.org/music/artist/prince/albums/1999/Little%20Red%20Corvette.wav

Content-Type: audio/vnd.wave
Content-Length: 70790540
POST /music/artist/prince/albums/1999/Little%20Red%20Corvette.wav`

Hypestore.org's Hypestore server receives the POST request, and stores the appropriate media asset in it's volume earmarked for storage (directly attached or "backless"). This is helpful, since the file will be available for Hypestore to respond with when the next GET /music/artist/prince/albums/1999/Little%20Red%20Corvette.wav comes around.
